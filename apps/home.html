Web-Based TI-84
     A web based reimagination of the Texas Instruments TI-84 Plus Silver Edition graphing calculator. The calculator was originally created as a Khan Academy project to supply a jumping off point for more complex HTML5 canvas projects, but soon grew into a graphing calculator.

     You can see the original Khan Academy version right here; it's not updated as often as the version hosted here, so it might not be fully up to date.

CSS Chat Bot
     Initially an idea for the front page of this site, this was meant as a interactive and amusing demonstration of web development. The project consists of a chat window through which you speak to AImy, your guide. She begins a conversation regarding the state of the web page and, using your input, begins to build a web page around her.

     Since this idea was never used, as you can tell by the existing, minimalist front page, the project was moved to its own repo for continued development as the silly gimmick that it is.

Crash Bandicoot Clock
     A countdown clock counting down towards the release date of the Crash Bandicoot N. Sane Trilogy, a long-awaited remake of the classes Crash Bandicoot games originally created for the PlayStation.

     That's pretty much it. Lettering was taken directly out of Crash Bandicoot 2 data files, making it more authentic, I suppose.
